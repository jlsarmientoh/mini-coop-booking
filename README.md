# mini-coop-booking

Vehicle reservation sample App

# Instructions

## Test the App

```npm test``` to run the unit tests and ```npm run it``` to run the integration tests

## Run the App

```npm install && npm start```

# Tech Stack

## BackEnd

- [NodeJS](https://nodejs.org/)
- [NestJS](https://nestjs.com/)

## FrontEnd

- [ReactJS](https://reactjs.org/)
